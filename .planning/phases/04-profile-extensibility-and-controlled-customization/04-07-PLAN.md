<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| UI controls → profile manager commands | User actions can import, save, export, delete, or remap profile data. |
| visual state → user decision | Source/editability/recovery state must be understandable without color-only cues. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-07-01 | Tampering | Profiles tab action buttons | mitigate | Bind buttons to validated ViewModel commands from Plans 05/06; UI does not write files directly. |
| T-04-07-02 | Spoofing | Source badges | mitigate | Display explicit text badges for Bundled/Custom/Embedded/Missing so custom data cannot appear bundled. |
| T-04-07-03 | Repudiation | Remap/recovery UI | mitigate | Recovery/remap controls route through undo-aware commands and visible neutral copy. |
</threat_model>
