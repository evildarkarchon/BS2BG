<threat_model>
## Trust Boundaries
| Boundary | Description |
|----------|-------------|
| Project/profile data→zip | Shared project data becomes an archive sent to another person. |
| Filesystem path→manifest/report | Local absolute paths may accidentally enter support artifacts. |

## STRIDE Threat Register
| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-06-01 | Tampering | Zip entries | mitigate | Normalize relative entry paths, reject rooted paths and .. segments, assert ZipArchive entries in tests. |
| T-05-06-02 | Information Disclosure | manifest/report | mitigate | Central BundlePathScrubber and tests scan for drive roots, backslashes, usernames, and original directories. |
| T-05-06-03 | Tampering | bundle target | mitigate | Refuse existing zip target unless explicit overwrite is true. |
</threat_model>
